Microsleep Detection App with Avatar AI

This app detects yawning, drowsiness, and microsleep using Google ML Kit. Integrated with an AI avatar chatbot that gives real-time alerts and safe-point suggestions. Developed using Flutter, TTS, Firebase, and Waze API.

IoT Microsleep Alarm for Driver

This system uses OpenCV and face landmarks to detect driver microsleep, triggering alerts via MQTT and Slack API. Designed for road safety using real-time IoT integration.
