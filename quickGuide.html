🌐 Apa Itu Network?
▼
Network ni macam jalan raya untuk komputer. Bila dua atau lebih komputer bersambung, dia boleh share info, file, internet, printer, etc.
Contoh: Laptop kau connect ke WiFi rumah - itu dah jadi satu network. Boleh share printer dengan PC lain.
📡 Jenis-jenis Network
▼
  • LAN (Local Area Network): Kecil je. Contoh: dalam rumah, sekolah, pejabat kecik.
  • WAN (Wide Area Network): Sambungkan banyak LAN dari tempat berbeza. Contoh: Internet.
  • PAN (Personal Area Network): Kecil dan peribadi. Contoh: Bluetooth phone ke headphone.
  • MAN (Metropolitan Area Network): Cover satu bandar. Macam network provider dalam satu kawasan.
🔌 Network Topologies
▼
Topology ni cara komputer arrange dalam network. Macam floor plan rumah, tapi untuk network.
  • Star: Semua connect ke satu central hub. Kalau hub rosak, semua mati.
  • Bus: Semua connect ke satu main cable. Murah tapi slow bila ramai.
  • Ring: Connect dalam bentuk bulatan. Data pergi satu arah je.
  • Mesh: Semua connect dengan semua. Mahal tapi reliable gila.
📚 OSI Model - 7 Layers
▼
OSI Model ni macam blueprint untuk networking. Ada 7 layer, setiap layer ada kerja sendiri.
Layer Nama Fungsi Contoh
7 Application Interface dengan user Browser, Email
6 Presentation Encryption, Compression JPEG, SSL
5 Session Manage connections SQL Sessions
4 Transport End-to-end delivery TCP, UDP
3 Network Routing, IP addressing Router, IP
2 Data Link Frame formatting Switch, MAC
1 Physical Physical connection Cable, Fiber
Memory trick: "Please Do Not Throw Sausage Pizza Away" - Physical, Data Link, Network, Transport, Session, Presentation, Application
🌍 TCP/IP Model
▼
TCP/IP ni model yang sebenarnya digunakan dalam internet. Lebih simple dari OSI - 4 layer je.
  • Application Layer: HTTP, FTP, SMTP, DNS
  • Transport Layer: TCP (reliable), UDP (fast)
  • Internet Layer: IP, ICMP, ARP
  • Network Interface: Ethernet, WiFi
🏠 IP Addressing & Subnetting
▼
IP Address macam alamat rumah untuk komputer. Ada IPv4 (32-bit) dan IPv6 (128-bit).
IPv4 Classes:
Class A: 1.0.0.0 - 126.255.255.255 (16 million hosts)
Class B: 128.0.0.0 - 191.255.255.255 (65k hosts)
Class C: 192.0.0.0 - 223.255.255.255 (254 hosts)
  • Private IP Ranges:
  • 10.0.0.0 - 10.255.255.255 (Class A)
  • 172.16.0.0 - 172.31.255.255 (Class B)
  • 192.168.0.0 - 192.168.255.255 (Class C)
🔄 Network Devices
▼
  • Hub: Basic repeater. Semua port dapat same data. Dah outdated.
  • Switch: Smart hub. Each port dapat data yang betul je. Layer 2 device.
  • Router: Connect different networks. Layer 3 device. Ada routing table.
  • Access Point: Wireless hub. Convert wired ke wireless.
  • Firewall: Security guard. Filter traffic masuk keluar.
📨 Common Protocols
▼
Protocol Port Fungsi
HTTP 80 Web browsing
HTTPS 443 Secure web browsing
FTP 21 File transfer
SSH 22 Secure remote access
SMTP 25 Send email
DNS 53 Domain name resolution
DHCP 67/68 Automatic IP assignment
🛡️ Network Security Basics
▼
  • Firewall: Block unwanted traffic. Macam bouncer club.
  • VPN: Secure tunnel through internet. Macam terowong rahsia.
  • WPA/WPA2: WiFi security. Jangan guna WEP, dah outdated.
  • MAC Filtering: Only allow specific devices. Tapi boleh spoof.
  • Network Segmentation: Pisahkan network untuk security.
